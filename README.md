# Titanic Survival Prediction Using Logistic Regression

## Project Overview
The sinking of the Titanic on April 15, 1912, is one of the most infamous maritime disasters in history. While survival depended on several factors, certain groups of passengers were more likely to survive than others.  

This project aims to **predict passenger survival** using logistic regression based on features such as age, gender, passenger class, and other socio-economic factors.

## Dataset
The dataset includes historical passenger information from the Titanic, with features such as:
- **PassengerId** – Unique ID for each passenger  
- **Survived** – Target variable (0 = No, 1 = Yes)  
- **Pclass** – Passenger socio-economic class  
- **Name** – Name of passenger  
- **Sex** – Gender of passenger  
- **Age** – Age in years  
- **SibSp** – Number of siblings/spouses aboard  
- **Parch** – Number of parents/children aboard  
- **Ticket** – Ticket number  
- **Fare** – Passenger fare  
- **Cabin** – Cabin number  
- **Embarked** – Port of embarkation  

## Objectives
- Analyze the relationship between passenger features and survival.  
- Build a **logistic regression model** to predict survival.  
- Evaluate model performance using:
  - Accuracy
  - Precision
  - Recall
  - F1 Score
- Visualize survival predictions and feature importance.
